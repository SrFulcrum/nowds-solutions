Site institucional da Nowds Solutions.

Arquivos:
- index.html
- Dockerfile

Como usar no EasyPanel:
1. Envie esta pasta para um repositório no GitHub.
2. No EasyPanel, crie um App e conecte o repositório.
3. Como o projeto já tem Dockerfile, o EasyPanel vai buildar a imagem.
4. A porta interna do container é 80.
5. Depois configure o domínio e faça o deploy.
